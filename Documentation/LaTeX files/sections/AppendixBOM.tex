\section{Bill of Materials}
\label{app:bom}

This appendix lists the BOMs for the PLC Lab Trainer System. The two trainer boxes share a common set of enclosure, cabling, and operator-interface parts (Section~\ref{sec:bom-common}); each box then adds its own subsystem-specific components (Sections~\ref{sec:bom-heater-specific} and~\ref{sec:bom-motor-specific}). Section~\ref{sec:bom-pico} lists the components on the Pico H-Bridge motor control board that mounts inside the Motor Control Box. Box-level quantities are listed per unit; the project deploys 16 of each box.
