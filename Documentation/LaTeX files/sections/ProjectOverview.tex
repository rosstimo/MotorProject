\section{Project Overview}

\subsection{Introduction}

This document describes the \textbf{PLC Lab Trainer System}, a modular educational platform developed at Idaho State University for teaching programmable logic controller (PLC) programming and process control. The system is built around a single Allen-Bradley CompactLogix PLC rack and a family of swappable lab trainer boxes that share one common electrical interface. Each box presents a different control problem so students can progress from basic discrete I/O up to closed-loop motion control without rewiring the PLC panel.

Two trainer boxes are documented here:

\begin{itemize}
    \item The \textbf{Heater Control Box} --- an introductory thermal trainer that exercises digital I/O, analog input scaling, and on/off or PID temperature control.
    \item The \textbf{Motor Control Box} --- an advanced motion trainer built around a Raspberry Pi Pico and an H-Bridge that drives a DC brushed motor with quadrature encoder feedback.
\end{itemize}

Both boxes connect to the PLC through a shared \textbf{DB37 passthrough interface board}. The passthrough breaks out the 37-conductor cable from the PLC wiring panel to Phoenix Contact screw terminals, so any compliant box can be wired in by terminating to those terminals. Power rails (+24\,V, +12\,V, GND), digital I/O, and analog I/O are all carried on the same DB37 connector regardless of which box is attached. Figure~\ref{fig:overall-concept} shows this overall architecture.

\subsection{Learning Progression}

The two boxes are designed to be used in sequence. Students start with the Heater Control Box, where the plant dynamics are slow, signals are simple, and a textbook PID loop produces visibly correct behavior. Once they are comfortable with PLC programming, analog scaling, and tuning, they move to the Motor Control Box, where signals are faster, feedback is multi-variable (speed and position).
