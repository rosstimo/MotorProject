    \subsection{TS-25GA370H-20 Motor}
    Encoder Metal Gearmotor 12V DC High Speed 300RPM Gear Motor with Encoder for Arduino and 3D Printers\\

    \noindent The test motor was purchased here: \href{https://www.amazon.com/Bemonoc-Encoder-Gearmotor-Two-Channel-Effect/dp/B07GNGCNK1?th=1}{www.amazon.com (Encoder Metal Gearmotor 12V DC High Speed 300RPM Gear Motor with Encoder for Arduino and 3D Printers)}

    \noindent \textbf{Product type}: DC Gear motor with two-channel Hall effect encoder
    \begin{itemize}
        \item Rated Voltage: 12V
        \item No-Load Speed: 300RPM
        \item No-Load Current: $\leq$ 0.15A
        \item Rated Torque: 0.5kg.cm
        \item Single Output 240 Pulses Per Revolution
        \item Gear Reduction Ratio: 1/20
        \item Each Loop Output Pulses: 12PPR 20$\times$12=240PPR
    \end{itemize}
